Calculadora

Este é um programa Python que solicita ao usuário um número e exibe a tabuada desse número de 1 a 10. Ele verifica se o número inserido é negativo e pede uma entrada válida, se necessário. Clique aqui para conferir o código fonte no GitHub.

Jogo de Advinhação

Jogo de adivinhação em Python onde o jogador tenta adivinhar um número aleatório entre 0 e 10. O programa oferece feedback sobre os palpites e informa quantas tentativas foram necessárias para acertar. Clique aqui para conferir o código fonte no GitHub.

Pedra, Papel e Tesoura

Jogo de pedra, papel e tesoura em Python onde o usuário joga contra o computador. O programa escolhe aleatoriamente um dos itens e verifica o resultado da partida. Clique aqui para conferir o código fonte no GitHub.

Contador de Passos

Utilizando a linguagem JavaScript, o projeto é composto por um contador de passos que conta de forma crescente ou decrescente com base nos parâmetros fornecidos pelo usuário. A contagem é exibida com emojis indicando a direção. Clique aqui para conferir o código fonte no GitHub.

Verificador de Idade

Um modelo que utiliza HTML, CSS e a linguagem JavaScript para verificar a idade com base no ano de nascimento e no sexo fornecido pelo usuário. Ao clicar no botão "Verificar", o resultado é exibido. Clique aqui para conferir o código fonte no GitHub.
